.NET sendiri merupakan software development platform
 yang menyediakan berbagai framework atau kerangka kerja
 yang disediakan oleh Microsoft untuk pengembangan perangkat lunak.

Sebagai platform, .NET menyediakan beberapa kerangka kerja untuk membuat berbagai jenis
 aplikasi seperti aplikasi web, mobile, desktop, sampai dengan aplikasi IoT atau Artificial Intelegence.

Misalnya ketika memilih ASP.NET sebagai kerangka kerja, kita hanya bisa mengembangkan aplikasi berbasis
 web seperti mengembangkan Web API. Ketika memilih Xamarin, kita hanya bisa mengembangkan aplikasi berbasis mobile. 
Begitu juga ketika memilih Windows Form atau WPF, kita hanya bisa mengembangkan aplikasi desktop di sistem operasi Windows.

 .NET Core.

Sejak itu, dengan menggunakan Mono for Xamarin sebagai kerangka kerja, kita bisa mengembangkan aplikasi native untuk Android dan 
iOS dengan kode program dari bahasa pemrograman yang sama (C# adalah bahasa pemrograman yang umum digunakan).
.NET memiliki dua komponen utama, yaitu sebuah mesin virtual bernama Common Language Runtime (CLR) dan juga kumpulan pustaka 
(toolbox) bernama Base Class Library (BCL) untuk .NET Framework atau Core Library untuk .NET Core.
